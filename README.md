## Run

Clone repository

```
git clone https://github.com/dayknightmare/portfolio.git

OR

git clone git@github.com:dayknightmare/portfolio.git
```
Install dependencies

```
npm i
```
Run dev mode

```
npm run dev
```

Open [Localhost:3000](http://localhost:3000) on your browser
